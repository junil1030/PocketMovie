## 🔧 핵심기능
* `일간/주간 박스오피스` 박스오피스 정보 열람
* `영화 검색` 영화 검색 기능
* `영화 기록` 평점, 한줄평 저장 기능
* `카드 덱` 수집한 영화를 카드 형태로 저장
